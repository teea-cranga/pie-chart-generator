About the site

This site is a project for the Multimedia class from our university and it was made by Cranga Ana-Teea and Ispas Anisia.

What does it do?

The site generates pie charts with labels using .csv files the user adds as input. The user can also save the pie chart as a .png file if needed.

Technologies used:

The site makes use of the Bootstrap framework, and the javaScript API's Drag & Drop and Canvas.
